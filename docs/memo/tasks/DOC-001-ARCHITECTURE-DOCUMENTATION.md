## 🏗️ 基础信息

### 新架构核心组件

基于 MIGRATE-001 的成果，新架构包含以下核心组件：

1. **TooltipManager** - 主管理器，单例模式
2. **TooltipStateManager** - 状态管理和事件监听
3. **TooltipEventHandler** - 事件处理系统
4. **TooltipPositioner** - 智能位置计算
5. **TooltipRenderer** - React组件渲染
