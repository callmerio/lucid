🎯 Lucid Tooltip 扩展功能测试

📋 新功能说明

Tooltip 现在支持右侧扩展功能，当鼠标悬停在 tooltip 右侧 2/5 区域时，会向右扩展并显示两个交互按钮。

🎯 测试步骤：

  1. 步骤1选择下面的英文单词，按 Shift 键进行高亮
  2. 步骤2将鼠标悬停在高亮的单词上，显示基础 tooltip
  3. 步骤3将鼠标移动到 tooltip 的右侧 2/5 区域（从 3/5 位置开始）
  4. 步骤4观察 tooltip 立即向右扩展并显示两个按钮
  5. 步骤5将鼠标移回左侧 3/5 区域，观察 tooltip 收缩
  6. 步骤6测试按钮功能：
    • 下三角按钮：展开详情（控制台日志）
    • 大拇指按钮：收藏单词（控制台日志 + 视觉反馈）
预期效果（最新优化）：
  • 鼠标移动到 tooltip 右侧 2/5 时，tooltip 立即平滑扩展（无延迟）
  • 显示两个按钮：下三角 ▼ 和爱心 ❤️（爱心缩小至 70%）
  • 按钮有弹性动画效果：从右侧滑入并带有缩放效果
  • 新增：hover 按钮时有心跳动画效果
  • 鼠标移回左侧 1/3 区域时，扩展状态收缩
  • 新增：按钮退出动画与整体 tooltip 退出动画同时发生
  • 整体高度保持不变，不会因为按钮出现而变高
  • 使用弹性缓动函数提供更自然的动画

🔍 调试信息

实现细节（已优化）：

  • 触发区域：tooltip 右侧 2/5 区域（从 3/5 位置开始，通过 mousemove 事件检测）
  • 扩展延迟：无延迟（立即响应）
  • 动画时长：400ms 平滑过渡（更慢更优雅）
  • 高度控制：固定高度 20px，防止按钮出现时变高
  • 按钮功能：下三角展开详情，大拇指收藏单词

检查方法：

  • 打开浏览器开发者工具的控制台
  • 观察按钮点击时的日志输出
  • 检查 tooltip 元素的 class 变化
  • 观察按钮的视觉反馈效果
  • 验证 tooltip 高度是否保持不变
